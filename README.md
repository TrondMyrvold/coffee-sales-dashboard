# Coffee Sales Performance Report Dashboard

## Project Overview

This project was developed in Power BI to demonstrate how sales data can be transformed into meaningful business insights through interactive dashboards. The dashboard is designed to help business owners monitor sales performance, identify customer purchasing patterns and support data driven decision making.

Rather than analysing raw transaction data, the dashboard provides an interactive overview of key business metrics that allow decision makers to quickly identify trends, compare product performance and explore sales behaviour across different time periods. Users can filter the dashboard by for example date and coffee type, making it easy to answer specific business questions and investigate changes in customer demand.

The dashboard demonstrates how Business Intelligence tools such as Power BI can support operational decision making by transforming large datasets into clear and actionable visual insights.

---

## Objectives

- Monitor total sales
- Compare coffee product performance
- Analyse sales over time
- Explore sales by weekday
- Analyse sales by time of day
